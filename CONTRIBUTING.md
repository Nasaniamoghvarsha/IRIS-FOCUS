# Contributing to Iris Focus

First off, thanks for taking the time to contribute!

## How Can I Help?

- **Reporting Bugs**: Use GitHub Issues.
- **Submitting Pull Requests**: 
    - Fork the repo.
    - Create a feature branch.
    - Ensure code passes linting (`npm run lint`).
    - Submit the PR.

## Code Standards
- Use TypeScript for all new logic.
- Follow the existing Tailwind CSS design system.
- Document all core functions with JSDoc.
- **Privacy First**: Any new feature must be strictly local-only. Never add external analytics or data-transmitting APIs.
